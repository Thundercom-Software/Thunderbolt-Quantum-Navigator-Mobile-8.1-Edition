# Thunderbolt-Quantum-Navigator-Mobile-8.1-Edition
The Windows Phone 8.1 port of Thunderbolt Quantum Navigator (Mobile)

Disclaimer:
This version of Thunderbolt Quantum Navigator (Mobile) is currently in the early stages of development and does not yet reflect the functionality or design of the final product. As development is still in its infancy, update releases may be less frequent than those for Thunderbolt Quantum Navigator (Mobile) for Windows 10 Mobile. Additionally, because Windows Phone 8.1 is a significantly older platform, even when compared to the final release of Windows 10 Mobile, development for this version is more complex and time consuming. As a result, progress may be slower while compatibility and stability are being addressed. Report any issues regarding this version of the browser in this repository.
